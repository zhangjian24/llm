## 📈 可靠性保证

### 标准化断言

```python
# ✅ 推荐：清晰的断言消息
assert result.status == "success", f"Expected success but got {result.status}"

# ❌ 避免：无消息的断言
assert result.status == "success"
```

### Mock 最佳实践

```python
# ✅ 正确：Mock 特定方法
mock_service.specific_method = AsyncMock(return_value=test_data)

# ❌ 避免：Mock 整个对象
mock_service = Mock()  # 可能遗漏重要方法
```

### 夹具层级管理

```python
# conftest.py - 全局夹具
@pytest.fixture(scope="session")
def app():
    """应用级夹具（整个测试会话共享）"""
    ...

# test_module.py - 模块级夹具
@pytest.fixture
def db_session():
    """会话级夹具（每个测试函数独立）"""
    ...
```
