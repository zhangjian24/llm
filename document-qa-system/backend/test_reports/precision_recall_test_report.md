## 附录 B: 核心公式

### Precision@K

$$\text{Precision@K} = \frac{\text{Top-K 结果中的相关文档数}}{K}$$

### Recall@K

$$\text{Recall@K} = \frac{\text{Top-K 结果中的相关文档数}}{\text{所有相关文档总数}}$$

### NDCG@K

$$\text{DCG@K} = \sum_{i=1}^{K} \frac{rel_i}{\log_2(i+1)}$$

$$\text{NDCG@K} = \frac{\text{DCG@K}}{\text{IDCG@K}}$$

其中 $rel_i = 1$ 如果第 i 个结果相关，否则为 0
