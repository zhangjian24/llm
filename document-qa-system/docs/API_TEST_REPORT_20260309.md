### 3.3 遗留问题说明

#### 问题 1: Pinecone 配置缺失导致 RAG 功能不可用

**问题描述**:
- 调用 `/api/v1/chat/` 接口时返回 500 错误
- 错误消息："文档检索失败"
- 根本原因：Pinecone Index 不存在或未正确配置

**影响范围**:
- FR-004: 智能问答生成功能无法使用
- FR-003: 语义检索功能无法验证
- FR-006: 流式对话功能无法验证

**规避方案**:
```bash
# 在 .env.local 中配置有效的 Pinecone 凭证
PINECONE_API_KEY=pcsk_your_key_here
PINECONE_INDEX_NAME=document-qa-index

# 或创建新的 Index
python create_pinecone_index.py
```

**优先级**: 🔴 高（影响核心 RAG 功能）
